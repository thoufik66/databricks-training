# Week 2 - Day 4: Numeric & Null Handling Functions

This day covers handling mathematical anomalies and building fail-safe calculation pipelines. You will practice null-safe evaluations and mathematical transformations.

## Topics Covered
* Null checks and logic (IS NULL, IS NOT NULL)
* Replacing null values with default metrics (IFNULL, COALESCE)
* Value suppression and preventing division by zero (NULLIF)
* Rounding and adjustments (ROUND, CEIL, FLOOR, TRUNCATE)
* Mathematical operators (MOD, POWER, GREATEST, LEAST, FORMAT)
