# Week 2 - Day 2: Date & Time Manipulations

This day focuses on working with temporal data. You will practice extracting date parts, conducting date math (adding/subtracting intervals), calculating age or durations, and formatting calendar values.

## Topics Covered
* Server date/time checks (CURDATE, NOW)
* Slicing components (YEAR, MONTH, DAY, EXTRACT)
* Calendar designations (MONTHNAME, DAYNAME)
* Weekday tracking indices (WEEKDAY, DAYOFWEEK)
* Date intervals (DATE_ADD, DATE_SUB, DATEDIFF)
* Formatting and type parsing (DATE_FORMAT, STR_TO_DATE)
* Dynamic corporate fiscal calendars (Financial Year calculation)
