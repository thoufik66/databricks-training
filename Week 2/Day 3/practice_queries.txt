Week 2 - Day 3: Advanced Joins & Set Operations

This worksheet contains table schemas, mock data, and 30 practice questions covering multi-table relationships, self-joins, emulating FULL OUTER JOIN in MySQL, set operations, and aggregation across joined tables.

--------------------------------------------------------------------------------
Table Schemas:
--------------------------------------------------------------------------------

CREATE TABLE departments (
    dept_id INT PRIMARY KEY,
    dept_name VARCHAR(50)
);

CREATE TABLE employees (
    emp_id INT PRIMARY KEY,
    emp_name VARCHAR(50),
    manager_id INT,
    dept_id INT,
    salary DECIMAL(10,2)
);

CREATE TABLE projects (
    project_id INT PRIMARY KEY,
    project_name VARCHAR(50),
    emp_id INT
);

CREATE TABLE employee_contacts (
    emp_id INT PRIMARY KEY,
    phone VARCHAR(20),
    email VARCHAR(50)
);
