# Week 2 - Day 6: String Functions in MySQL

This day is dedicated to standard string manipulation operations in MySQL. You will practice cleaning whitespaces, parsing email suffixes, padding indices, and searching arrays.

## Topics Covered
* Length auditing (LENGTH)
* Casing modifiers (UPPER, LOWER)
* Cleaning errant whitespaces (TRIM)
* Concatenations (CONCAT, CONCAT_WS)
* Character slicing (SUBSTRING, LEFT, RIGHT)
* Index lookup matches (INSTR, LOCATE, FIND_IN_SET)
* Character padding and reverse logic (LPAD, REVERSE)
