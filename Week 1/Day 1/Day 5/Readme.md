# Week 1 - Day 5 - Window Functions and CTEs

## Overview

Day 5 continued window functions and introduced CTEs (Common Table Expressions). Practiced on [db-fiddle.com](https://www.db-fiddle.com/) using a retail sales dataset.

---

## Topics Covered

- AVG() OVER(PARTITION BY) - department average salary alongside each row
- Filtering rows using window function results in a subquery
- SUM() OVER(PARTITION BY) - department payroll
- Percentage contribution of salary within department
- COUNT() OVER() - total row count alongside each row
- CTE (WITH clause) - reusable named subquery
- Multiple CTEs in a single query
- Recursive CTE - generating number sequences
- Recursive CTE - employee hierarchy traversal
- Combining CTEs with window functions
