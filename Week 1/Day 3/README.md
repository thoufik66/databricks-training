# 📅 Week 1 — Day 3: SQL Joins Assignment

> **Focus:** Deep dive into SQL JOIN variations (LEFT, RIGHT, FULL, CROSS) and logical operators using a multi-table educational database schema.

---

## 📝 Overview

Day 3 assignment on SQL JOINs using a simple online learning platform dataset — Students, Courses, Enrollments, and Instructors. All queries were practiced on [db-fiddle.com](https://www.db-fiddle.com/).

---

## 📚 Topics Covered

- **LEFT JOIN** & **RIGHT JOIN** to retrieve matching data along with unmatched rows from either side
- **FULL OUTER JOIN** to combine left and right outer joins
- **CROSS JOIN** to produce a Cartesian product of both tables
- **Identifying Unmatched Rows** using `IS NULL` filters
- **Chaining Multiple JOINs** to query across 3+ tables simultaneously
- **UNION** operators to combine results from multiple SELECT statements
